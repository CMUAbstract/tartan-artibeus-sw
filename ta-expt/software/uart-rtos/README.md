# Tartan Artibeus Experiment Board Blink RTOS Software

Tartan Artibeus experiment board blink RTOS software

```bash
cd ../../scripts/
source sourcefile.txt
cd ../software/uart-rtos/
make
st-flash write uart-rtos.bin 0x8000000
```
